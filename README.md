# Port-scanner
# Simple Port Scanner

A lightweight port scanner built with Python's standard library.

## Features
- Fast scanning using socket connections
- Timeout handling for unresponsive hosts
- Clean output format

## Installation
No installation required. Just clone the repo and run.

```bash
git clone https://github.com/man_dev/port-scanner.git
cd port-scanner
```

## Usage
```bash
python scanner.py <target_ip>
```

Example:
```bash
python scanner.py 192.168.1.1
```

## Note
This tool is for educational purposes only. Always get permission before scanning any network.
